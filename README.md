# Исправление русской локализации (исключительно ролевой системы) для игры «Pathfinder: Wrath of the Righteous»

Русская локализация игры [Pathfinder: Wrath of the Righteous](https://store.steampowered.com/app/1184370/Pathfinder_Wrath_of_the_Righteous__Enhanced_Edition/) содержит множество недостатков и проект является попыткой убрать неточности, ошибки и несогласованность с английской версией локализации и фактической механикой игры, которая основана на правилах настольной ролевой игры Pathfinder первой редакции (далее — pf1ed).

Проект создан исключительно для исправления ролевой системы (набора правил pf1ed) игры и ни в каком виде не затрагивает нарративную часть (диалоги, сюжет и прочие описания, не касающиеся ролевой системы).

## Установка

Установка возможна двумя способами:
1. Без использования модов и Unity Mod Manager (далее — UMM):
    - Скачайте файл `ruRU.json` и сохраните/скопируйте с заменой в папку с файлами локализаций игры
    - Например, для Steam эта папка будет находиться по пути `"{Путь_до_Хранилища_игр_Steam}\steamapps\common\Pathfinder Second Adventure\Wrath_Data\StreamingAssets\Localization"`
2. Через UMM при помощи мода [DynamicLocalizationLoader](https://github.com/alterasc/DynamicLocalizationLoader):
    - Скачайте и установите мод
    - Скачайте содержимое папки `DynamicLocalizationLoader\Packs` и сохраните в папке локализаций мода согласно его инструкций
